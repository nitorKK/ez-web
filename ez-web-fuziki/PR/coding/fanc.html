アプリの機能

このアプリ1つでレッスンがすべて完結します

お気に入りの先生を探すのも、スマホから書き込めるホワイトボードでの授業も、予約、授業、決済、復習まで全部が1つのアプリでできます。スマホアプリ1つで、全てが完結。あなたのポケットに、家庭教師を。

アプリの特徴

Feature 1

便利なホワイトボード機能

先生が書いたホワイトボードがリアルタイムで画面に表示されます。家庭教師専用アプリだから、メッセージ機能とホワイトボード、ビデオ通話が1つの画面で使えます。

Feature 2

自分に合った先生が見つかる

科目や時間帯、出身大学などから条件を絞り込み、先生のプロフィールや指導科目を確認できるので自分にあった先生を見つけることができます。

Feature 3

条件に合う先生とマッチング

先生は特に指定せずに、日時や科目など自分が設定した条件をリクエスト。条件に合う先生が見つかったらレッスンが受けられます。

Feature 4

ちょっとしたスキマ時間に一問一答

1問につき15分間以内で先生に質問することもできます。料金は一律900円。気軽にちょっと聞いてみたいときやスキマ時間に活用できます。先生は特に指定せずに、日時や科目など自分が設定した条件をリクエスト。条件に合う先生が見つかったらレッスンが受けられます。
